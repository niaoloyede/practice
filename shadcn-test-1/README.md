# shadcn-test-1

A **Next.js 16** app with [shadcn/ui](https://ui.shadcn.com) for testing and prototyping UI components.

## Stack

- **Next.js** (App Router), **React 19**, **TypeScript**
- **Tailwind CSS 4**, **shadcn** (Radix UI + Base UI)
- **Lucide React** for icons, **Geist** font via `next/font`

## What's included

- **UI components:** Button, Card, Badge, Input, Label, Textarea, Select, Combobox, Dropdown Menu, Alert Dialog, Separator, Field, Input Group
- **Example components:** theme toggle, component examples
- Dark/light theme support

## Getting started

