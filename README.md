# practice

A workspace for practice and experiments. Currently includes a Next.js app using [shadcn/ui](https://ui.shadcn.com) components.

## Projects

### shadcn-test-1

A **Next.js 16** app with **shadcn/ui** (Radix UI + Tailwind CSS) for testing and prototyping UI components.

- **Stack:** Next.js (App Router), React 19, TypeScript, Tailwind CSS 4, shadcn
- **Includes:** Example components (buttons, cards, forms, theme toggle), Base UI, Lucide icons

**Run it:**

```bash
cd shadcn-test-1
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.
